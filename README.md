# diy-smart-locker
Node.js + lockitron mechanical assembly
 
Lockitron mechanical assemblyをRaspberry PiのNode.jsから使ってみるデモ

## 使ったもの

- Lockitron mechanical assembly
- Raspberry Pi 2
- Node.js
  - pi-gpio
  - AngularJS
  
## 構成

Node.jsで立てたWebサーバーから鍵の状態を管理する。
